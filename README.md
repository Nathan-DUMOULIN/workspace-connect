# WorkSpace Connect - Site de Réservation de Salles

## 📝 Présentation du Projet
[cite_start]Ce projet a été réalisé dans le cadre de la **1ère année de BTS SIO (SLAM)**[cite: 2, 7]. 
[cite_start]**WorkSpace Connect** est une plateforme web permettant aux utilisateurs de consulter et de réserver des espaces de coworking et des salles de réunion en ligne[cite: 18, 21]. [cite_start]L'objectif est de remplacer la gestion manuelle actuelle par un système automatisé et performant[cite: 19].

## 🛠 Technologies Utilisées
* [cite_start]**Frontend :** HTML5, CSS3 (Responsive Design avec Media Queries) [cite: 32, 161]
* [cite_start]**Backend :** PHP (PDO pour la sécurité) [cite: 33, 191]
* [cite_start]**Base de données :** MySQL [cite: 34, 94]
* [cite_start]**Versionnage :** GitHub pour le travail collaboratif [cite: 35, 271]

## 📋 Fonctionnalités
* [cite_start]**Accueil :** Présentation de l'entreprise[cite: 25].
* [cite_start]**Catalogue :** Affichage dynamique des salles disponibles depuis la base de données[cite: 26, 197].
* [cite_start]**Détails :** Fiche complète de chaque salle (capacité, prix, équipements)[cite: 28, 156].
* [cite_start]**Réservation :** Formulaire complet avec enregistrement sécurisé en base de données[cite: 29, 218].
* [cite_start]**Confirmation :** Page de validation après chaque réservation réussie[cite: 73, 158].
